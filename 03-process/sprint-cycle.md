##Sprint Cycle

- Prioritize stories
- Prepare stories for development
  - Decompose into development tasks
  - Estimate tasks
  - Define acceptance criteria
- Implement story solutions
- Review/test implementation
- Demo/validate with users
- Repeat and repeat
